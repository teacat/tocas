Introduction

All items placed in this component are automatically separated by a blank space, which can be said to be a very brainless component.

If you want an item to fill the remaining space, please use Row. The item will automatically change to the next line when it is too wide.

Form Fields

The fields of the form are usually arranged vertically, so if you want to have a space between each field, putting them into a Wrap is the best choice.

Username
Password
<div class="ts-wrap is-vertical">
    <div class="ts-text is-label">Username</div>
    <div class="ts-input is-start-icon">
        <span class="ts-icon is-user-icon"></span>
        <input type="text" />
    </div>
    <div class="ts-text is-label">Password</div>
    <div class="ts-input is-start-icon">
        <span class="ts-icon is-lock-icon"></span>

</div>
