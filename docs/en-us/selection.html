Introduction

When using this component, we recommend having at least one active item by default to avoid users not knowing that it is an interactive form control component.

This component is used to switch back and forth between modes or options, if you wish to switch content in similar way please use Tab.

States
Disabled

Makes the Selection appear uninteractive and unclickable. If you want to actually disable the Selection behavior, make sure to add a disabled attribute to the Selection.

<div class="ts-selection">
    <label class="item is-disabled">
        <input type="radio" name="version" />
        <div class="text">Standard</div>
    </label>
    <label class="item">
        <input type="radio" name="version" checked />
        <div class="text">Advance</div>
    </label>
    <label class="item">
        <input type="radio" name="version" />
        <div class="text">Test</div>
    </label>
</div>

Fluid

The width of the Selection can be equaled to its parent container.

<div class="ts-selection is-fluid">
    <label class="item">
        <input type="radio" name="language" />

    </label>
    <label class="item">

        <div class="text">English</div>
    </label>
    <label class="item">
        <input type="radio" name="language" />
        <div class="text">Japanese</div>
    </label>
</div>
