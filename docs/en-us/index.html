No jQuery dependency.
In addition to consuming excess network bandwidth; jQuery is no longer the best choice for modern websites. Tocas UI works well with all front-end frameworks.
No need for npm or Webpack.
You don't need to install a bunch of packages via npm, nor setting up a bunch of loaders in Webpack. You just copy the HTML tags we provided and... TADA! It's that easy.
Full documentation.
Even if you haven't written Bootstrap or Bulma, Semantic UI, Material Design... you can still understand the whole interface system and how responsive design works with a complete explanation.
The design is like a whole new generation.
The existing components and styles are all available and ready for you.

Ready?
Copy the following HTML tags to your web page and have a blast.
<!-- Tocas UI：CSS 與元件 -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/tocas-ui/4.0.0/tocas.min.css">
<!-- Tocas JS：模塊與 JavaScript 函式 -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/tocas-ui/4.0.0/tocas.min.js"></script>
